# Objective

Build a world-class personal website for Swapnil Kumar.

The website must be completely static and deployable on GitHub Pages.

No backend.

No Node.js runtime required.

No databases.

Everything should work using HTML, CSS, JavaScript, Markdown, JSON and GitHub APIs.

The website should resemble premium developer portfolios such as:

- Vercel
- Stripe
- Linear
- Apple
- Anthropic
- OpenAI

The design should be elegant, premium, minimal and modern.

The site should focus on engineering excellence rather than flashy animations.

---

# Technology Constraints

Must use

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Markdown
- JSON
- GitHub Pages compatible

Optional

- AOS
- Chart.js
- FontAwesome
- Highlight.js
- Typed.js

No React.

No Angular.

No Vue.

No backend.

No server-side rendering.

No PHP.

No database.

No build step.

Everything should run by opening index.html.
